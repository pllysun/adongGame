# 02 · 修炼与境界

数值系统的心脏。所有经营行为（建设施、抢功法、炼丹药）最终都汇入两个数字：
**修为增速** 和 **突破成功率**。

## 1. 境界阶梯

九大境界，每境分初/中/后期三小层。寿元在**晋入大境界时**刷新。

| 境界 | 寿元上限 | 全程预期耗时（玄品资质·正常供养） | 定位 |
|------|---------|--------------------------------|------|
| 凡人 | 80 | — | 候选池 |
| 炼气 | 120 | 10~25 年 | 外门 |
| 筑基 | 250 | 40~80 年 | 内门，第一道天堑 |
| 金丹 | 500 | 100~150 年 | 宗门中坚，前期里程碑 |
| 元婴 | 1000 | 200~300 年 | 镇宗底蕴，始有天劫 |
| 化神 | 2000 | 300~500 年 | 一方巨擘 |
| 炼虚 | 3500 | 500 年+ | 世界级事件参与者 |
| 合体 | 5000 | — | 后期内容 |
| 大乘 | 8000 | — | 后期内容 |
| 渡劫 | — | 胜利条件层 | 渡九重天劫，飞升候选 |

> 节奏锚点（2026-06 无头模拟校准后的实测值，6 局 × 1500 年，自动玩家盲玩）：
> 首个金丹 ~40 年，元婴 ~35-50 年（祖师线），化神 ~100 年，炼虚 ~230 年，
> 合体 ~530 年，大乘 ~670 年，**首个渡劫 ~940 年（5/6 局达成）**。
> 自动玩家不会重点培养王牌，真人玩家集中投喂丹药/功法预计提前 20-30%。
> 校准期间新增两个系统性机制：**宗门底蕴**（镇宗最高境界每级全宗修炼 +4%，强宗出强徒）
> 与**威胁锚定**（敌人强度锚定宗门最强三人战力而非随年代膨胀，见 events/menace.ts）。

## 2. 修为增长模型

每名弟子有修为值，每日结算：

```
日增修为 = 基础速率(当前小层)
         × 灵根纯度系数 (0.5 + 纯度 × 0.1)
         × 功法系数 (品阶基数 × 五行匹配度)
         × 设施倍率 (聚灵阵全局 × 洞府个人)
         × 行为系数 (修炼 1.0 / 闭关 1.5 / 执勤 0.2 / 游历 0.6+奇遇)
         × 状态系数 (心境/伤势/丹药 buff)
```

- 基础速率随小层递增设计成**几何增长**（每大境界约 ×8），保证低阶功法/设施在高境界
  自然"不够用"，逼出持续的资源升级需求。
- 修为满 → 进入**瓶颈态**：小层瓶颈自动尝试（高成功率）；大境界瓶颈必须走突破流程。

## 3. 突破判定（游戏最核心的一次掷骰）

```
最终成功率 = 基础率(资质品级, 目标境界)
           × (1 + Σ加成) ，夹在 [1%, 95%]

Σ加成 = 丹药(突破丹按品阶 +10%~+40%，每境界首次有效)
      + 心境(圆满 +15% / 郁结 -10%)
      + 悟性顿悟(突破瞬间检定，悟性×2% 概率触发，触发则 +25%)
      + 地点(灵穴洞府 +5%，祖师护道 +8%)
      + 功法契合(主修功法品阶 ≥ 目标境界需求 +5%，低于则 -15%)
```

## 4. 天劫（元婴及以上的大境界突破附带）

突破判定成功后**紧接天劫战**：天雷按目标境界给战力数值，弟子以自身战力+法宝+宗门支援硬抗
（结算规则复用战斗系统，见 05）。

- 宗门可投入：护法大阵开启（耗灵石）、长老结阵分担、渡劫法宝。
- 天劫失败 = 重伤或陨落（不再走失败结果表）。
- 设计意图：让高境界突破变成**全宗门动员的事件时刻**，而不是一个人的掷骰。
- 渡劫期的"九重天劫"是终极版本：事件链形式，跨数年，期间敌宗可能趁机来犯——胜利前的最高潮。

## 5. 寿元与世代设计

- 寿元尽前 10 年进入"灯枯"状态：全属性衰减，触发"最后一搏"抉择事件
  （强行冲击突破：成功续命、失败立刻坐化 / 安排后事：传功弟子，关系网定向增益）。
- **传功**：老死前可将毕生修为的一部分灌给指定弟子（折损率高），
  让老一辈的死亡对宗门有正向遗产，世代之间有传承感。

## 6. 功法系统（修为模型的主要外部变量）

```ts
interface GongFa {
  id: string
  name: string          // 如《青木长生诀》
  rarity: Rarity        // 凡阶~仙阶
  element: Element[]    // 五行属性，决定与灵根的匹配度
  speedBase: number     // 修炼速度基数（随品阶几何增长）
  realmCap: Realm       // 功法支撑的境界上限（低阶功法修不到高境界）
  combatMod: number     // 战力修正
  special?: TraitTag[]  // 特效标签：如 魔道（杀伐加成/心魔风险+）、剑修（战力++修炼-）
}
```

- 每名弟子**主修一部**功法（决定速度与战力主成分），换功法有修为折损。
- `realmCap` 是关键钩子：黄阶功法只能修到金丹——你的天品天才拿不到天阶功法就是浪费，
  **抢功法因此成为中后期最强驱动力**（秘境/拍卖/宗门战战利品）。
- 获取途径：开局祖传 2~3 部、秘境、坊市拍卖、敌宗战利品、后期悟性 10 的弟子可自创（事件链）。
