# 05 · 战斗系统

定位：**战斗是故事生成器，不是操作挑战**。玩家的博弈全部在战前（派谁、带什么、要不要打）；
结算全自动，输出一段叙事化战报；伤亡与恩怨写回关系网。

## 1. 战力模型

```
个体战力 = 境界基数(大境界几何增长 ×6, 小层 ×1.15)
         × (1 + 功法战力修正 + 法宝加成 + 战斗经验等级 × 0.05)
         × 状态系数(伤势/心境/丹药)

队伍战力 = Σ个体 × 协同系数(1 + 0.05 × 平均关系强度档) × 阵法加成
```

- 境界差是压倒性的（设计意图：金丹镇压一群炼气是世界观常识），
  但**同境界内**功法/法宝/经验/协同的差距足以翻盘。
- 协同系数让"派一队挚友"优于"派一队仇人"——关系网反向影响战斗，闭环。

## 2. 结算流程

```
1. 力量对比 R = 我方战力 / 敌方战力
2. 胜负判定：P(胜) = R² / (R² + 1)   （平滑 S 曲线，R=1 时五五开）
3. 节拍模拟：3~6 个战斗节拍（beat），每拍从节拍库按局势抽取
4. 伤亡结算：败方按力量差距掷伤亡表；胜方也可能掷（惨胜）
5. 关系写回 + 战利品 + 声望结算 + 战报渲染
```

### 节拍库（beat templates）

节拍是战报的叙事原子，按局势标签（占优/胶着/逆风/转折）分库：

| 节拍类型 | 触发倾向 | 关系网效果 |
|---------|---------|-----------|
| 祭宝压制 | 占优方有高阶法宝 | — |
| 舍身相救 | 队内有挚友/道侣/倾慕边 | 被救者 +30 好感，可能升级关系 |
| 见死不救 | 队内有仇怨边 + 凉薄特质 | 结仇加深，事后人事事件 |
| 临阵顿悟 | 悟性 ≥ 8，小概率 | 战后修为暴涨，"战斗悟道"佳话 |
| 杀红了眼 | 杀伐特质 | 击杀率 +，战后心境检定 |
| 断后 | 溃败局 + 坚毅/侠义 | 高死亡率 + 高声望，悲壮节拍 |

节拍抽取结果**真实参与伤亡结算**（舍身相救者承担转移伤害），不是纯演出——
战报里的每句话都对应一次真实掷骰。

## 3. 伤亡表

| 结果 | 说明 |
|------|------|
| 轻伤 | 修炼效率 -20%，1~6 月恢复 |
| 重伤 | 停止修炼，需疗伤丹，1~3 年；落下暗伤（突破 -3%，可丹药根除） |
| 濒死 | 即时救治检定（队友医术/丹药），失败 → 阵亡 |
| 阵亡 | 葬礼事件，关系涟漪，遗物回收 |
| 被俘/被掳 | 进入营救事件链 |

死亡保护阀：**真传及以上弟子**首次濒死必触发"护道法宝碎裂/长老感应驰援"挽救事件
（每人一次）——防止玩家数百年心血死于一次低概率骰运，同时制造高光剧情。

## 4. 战报渲染

节拍序列 → 文本模板（带角色名/功法名/法宝名插值）→ 2~6 段战报文学。

> 示例输出：
> 北山妖兽潮压境，护山大阵嗡鸣不止。李青莲祭出青萍剑直取虎妖首级，剑光过处血雨纷飞。
> 激战正酣，三阶铁背狼突袭阵脚，王二狗推开师妹周芷若，自己被狼爪撕开胸膛——重伤垂死。
> 周芷若目眦欲裂，竟于生死一线间触机顿悟，一剑枭首。
> 此役大胜，斩获妖兽材料颇丰。王二狗重伤，周芷若对其好感大增。

UI 上战报逐段淡入，关键词按七阶颜色高亮（人名按资质色、法宝按品阶色）。

## 5. 战斗类型

| 类型 | 规模 | 特殊规则 |
|------|------|---------|
| 切磋 | 1v1 | 不死人（走火入魔小概率例外），产战斗经验与关系变化 |
| 秘境遭遇 | 小队 vs 妖兽/敌修 | 标准结算 |
| 妖兽潮 | 全宗防御战 | 护山大阵倍率参战；设施可被损毁 |
| 斗法 | 3v3 点将 | 敌宗寻衅的文明解法，败方损声望/资源 |
| 宗门战 | 多波次事件链 | 前哨战→阵法攻防→决战，跨数月，期间可求和/求援 |
| 天劫 | 1 v 天 | 渡劫者 + 宗门支援 vs 固定数值天雷，见 02 文档 §4 |

## 6. 玩家的战前界面（博弈所在）

出战配置界面 = 弟子多选列表 + 实时显示：预估双方战力比、胜率区间（模糊显示："胜算颇大/五五之数/凶多吉少"）、
队伍协同提示（"队中有道侣：协同+"／"李四与张三有仇：协同-"）。
**不显示精确百分比**——保留不确定感，鼓励留余量派人，败仗才会发生，故事才有起伏。
