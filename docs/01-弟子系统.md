# 01 · 弟子系统

弟子是游戏的最小叙事单元。设计目标：**每个弟子在数据上是几十个字段，在玩家眼里是一个人**。

## 1. 弟子生成

入门时生成以下字段：

| 字段 | 生成规则 |
|------|---------|
| 姓名 | 姓氏池（百家姓加权）+ 名字池（按性别/气质分库），保证同宗不重名 |
| 性别 / 入门年龄 | 招收大会以 8~16 岁少年为主；散修来投可以是成年人（自带境界） |
| 资质（七品） | 见 §2，候选池分布受声望影响 |
| 灵根 | 五行一至五属性组合 + 纯度 1~10；属性数越少纯度上限越高 |
| 悟性 / 心性 | 1~10 正态分布，与资质品级弱正相关 |
| 气运 | 1~10 **隐藏**，永不直接展示 |
| 性格特质 | 从特质库抽 2~4 条，互斥对不共存；部分隐藏 |
| 立绘种子 | 决定纸娃娃图层组合，外貌与气质字段挂钩 |

## 2. 资质七品

资质决定**突破概率曲线**（具体矩阵见 02 文档），是"能走多远"的主轴。

| 品级 | 颜色 | 招收大会出现率 | 备注 |
|------|------|--------------|------|
| 凡品 | 白 | 50% | 外门基本盘、宗门劳动力 |
| 灵品 | 绿 | 30% | 中坚执事层 |
| 玄品 | 蓝 | 13% | 内门主力 |
| 地品 | 紫 | 5% | 真传候选 |
| 天品 | 金 | 1.5% | 出现即宗门大事，全宗倾斜 |
| 圣品 | 红 | 0.45% | 数百年一遇 |
| 仙品 | 彩 | 0.05%（实际上几乎只出自事件链） | 神话级 |

- 声望提升候选池**数量**与凡→紫的分布，**天品以上出率基本不受声望影响**（防止后期招收变成刷卡）。
- **资质觉醒**：隐藏特质"大器晚成"使表显资质低一品，触发觉醒事件后跳升（最多 +2 品）。
- **伤仲永**：隐藏特质，长期心境受挫后实际曲线按低一品计算，直到通过事件解开心结。

## 3. 五维分工

| 维度 | 管什么 | 主要消费方 |
|------|--------|-----------|
| 资质（七品） | 走多远——突破概率曲线 | 突破判定 |
| 灵根 | 走多快——修炼速度 × 功法五行匹配 | 修为增长 |
| 悟性 | 学多快——功法掌握速度、顿悟概率 | 藏经阁、突破 |
| 心性 | 走多稳——心魔抵抗、走火入魔率 | 心境系统、突破失败结果表 |
| 气运（隐藏） | 多奇遇——个人事件权重系数 | 事件系统 |

信息迷雾：**资质、灵根、悟性**在测灵时可见；**心性**只能通过事件表现推断；**气运**永远隐藏。

## 4. 性格特质库（首批 22 条）

特质影响三处：①自主行为权重 ②专属事件解锁 ③心境与关系修正。V＝可见，H＝隐藏（事件中暴露）。

| 特质 | 对 | 可见 | 机制效果（示意） |
|------|----|------|----------------|
| 勤勉 | 懒散 | V | 修炼时间 +20% / 摸鱼事件，修炼效率 -15% |
| 坚毅 | 怯懦 | V | 突破失败心境损失减半 / 遇险逃跑，战斗士气 -20% |
| 痴情 | 凉薄 | H | 道侣关系成长 ×2，道侣亡故必触发心魔 / 几乎不结深交，心魔抗性 +30% |
| 侠义 | 阴鸷 | V | 救援事件主动参与，宗门声望 + / 结仇后解锁复仇事件链，魔道倾向 |
| 豁达 | 记仇 | V | 负面关系自然衰减 ×2 / 仇怨永不衰减，仇人突破时心境 - |
| 谦逊 | 傲慢 | V | 师徒关系成长 + / 被低品级同门超越时心境 -，切磋事件 + |
| 淡泊 | 贪婪 | H | 资源待遇不影响心境 / 月俸低时触发偷窃/叛逃检定 |
| 慈悲 | 杀伐 | V | 战斗不补刀，俘虏事件 + / 战报击杀率 +，魔道功法修炼加成 |
| 八面玲珑 | 孤僻 | V | 关系形成速度 ×2 / 几乎不主动社交，但修炼专注 +10% |
| 忠义 | 野心 | H | 叛逃检定永不通过 / 长老职位空缺时触发夺权事件链 |
| 武痴 | — | V | 演武场事件 ×2，战斗经验 +30%，社交事件 -50% |
| 大器晚成 | 伤仲永 | H | 见 §2 资质觉醒 / 见 §2 |

> 内容生产说明：特质是纯数据（JSON 条目 = 修正器列表 + 事件标签），加新特质零代码。
> 首发目标 30+ 条，上表为锚点示例。

## 5. 心境系统

心境是一个 -100~+100 的滑动值，五档状态，是**突破成功率的核心变量之一**，
也是性格/关系系统向数值系统传导的总管道。

| 区间 | 状态 | 效果 |
|------|------|------|
| +60↑ | 道心圆满 | 突破 +15%，顿悟概率 ×2 |
| +20~59 | 平静 | 无修正 |
| -19~+19 | 烦躁 | 修炼效率 -10% |
| -59~-20 | 郁结 | 修炼效率 -25%，突破 -10%，负面事件权重 + |
| -60↓ | 心魔缠身 | 禁止突破，每月走火入魔检定，触发心魔事件链 |

心境变动源：突破成败、亲近之人亡故/晋升、待遇变化（×贪婪/淡泊修正）、
事件选项、丹药（静心丹）、设施（静室）。自然回归速率受心性影响。

## 6. 关系网

### 数据形态

关系是**有类型、有强度、有历史的边**：`{ from, to, type, value(-100~100), history[] }`。

| 类型 | 形成途径 | 主要效果 |
|------|---------|---------|
| 师徒 | 拜师仪式（玩家规则或长老自动收徒） | 徒弟修炼 +，师亡徒心境 -，尊师/弑师事件 |
| 同门 | 同期入门自动建立弱边 | 协同作战小加成 |
| 挚友 | 共同历险/切磋/赠礼累积 | 心境互相传导，殉义救援事件 |
| 道侣 | 倾慕双向 + 事件确认（可被门规禁止） | 双修修炼加成，一方亡故另一方大概率心魔 |
| 倾慕 | 单向，性格驱动随机 + 事件 | 单恋心境波动，三角关系事件 |
| 仇怨 | 冲突事件/战斗见死不救/夺宝 | 切磋下死手、复仇事件链、叛逃诱因 |

### 性能纪律（涌现不等于全连接）

- 只存**强度越过阈值的边**，每名弟子有效边软上限 ~12 条，最弱边被新边挤出。
- 关系形成检定只在**社交池**内发生：同期、同设施共事、同队出任务、师徒链。
  几百弟子也不会 O(n²)。

## 7. 自主行为模型（间接管理的落地）

每名弟子每旬做一次**行为选择**，从可用行为中按权重抽取：

```
权重(行为) = 基础权重 × 性格修正 × 需求修正 × 宗门规则修正
```

- 行为池：修炼 / 演武 / 执勤（灵田、炼丹房等岗位）/ 社交 / 闭关冲击瓶颈 / 游历……
- **宗门规则**是玩家的主要控制面：如"真传弟子免执勤""金丹以下禁止游历"
  "突破前必须申请闭关"。规则改权重，不直接下命令。
- 玩家对单个弟子的直接干预仅限关键节点：批准/驳回闭关申请、赐丹药法宝、
  指派护道长老、收徒/逐出。

## 8. 弟子生命周期

```
入门（外门）→ 晋升（内门 → 真传 → 长老 → 掌门）
            → 离场：老死（寿元尽）/ 战死 / 走火入魔 / 叛逃 / 被掳 / 逐出
            → 身后：葬礼事件（关系网心境波动）、遗物入库、编年史立传
```

死亡不是数据删除而是**叙事节点**：关系网越深的弟子之死，涟漪越大。
