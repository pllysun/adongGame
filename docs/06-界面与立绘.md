# 06 · 界面与立绘

视觉基调：**水墨古风 + 信息密度克制**。深色底、宣纸纹理点缀、七阶颜色作为唯一的强彩来源
（稀有度色既是信息也是装饰）。

## 3. 纸娃娃立绘规格

### 图层栈（自底向上）

```
1. 底身（性别 × 体型 × 年龄段：少年/青年/中年/老者 —— 境界驻颜：金丹后外貌锁定青年）
2. 服饰（按身份：外门灰袍/内门青袍/真传纹袍/长老法袍 × 宗门主色）
3. 发型（×发色，老年灰白）
4. 眉眼 / 口鼻（各若干款，决定气质：英气/温婉/阴郁/憨厚……与性格标签弱关联）
5. 配饰（法宝佩剑/拂尘/玉佩 —— 反映装备）
6. 境界特效（筑基淡纹 → 金丹身周微光 → 元婴虚影 → 化神以上氛围粒子）
7. 边框（资质七阶色；仙品流光渐变动效）
```

- 每名弟子的图层组合由**立绘种子**决定，入门时固定，随身份/年龄/境界换层。
- 表情系统第一版只做 4 态：常态/喜/怒/伤，事件弹窗按情绪标签切换。

### 素材生产策略

1. **M0~M5 用程序化占位**：纯 SVG 色块小人 + 资质色边框 + 文字标签，先跑通全部逻辑。
2. **M6 接入正式素材**：AI 生成各图层（统一画风 prompt 模板）→ 人工筛选 → 切层入库。
   每层 8~12 款即可组合出数千形象。
3. 图层规格：1024×1536 PNG，锚点统一，命名约定 `{layer}_{variant}_{tag}.png`。

## 4. 编年史（叙事系统的收口）

引擎自动把**重大节点**写入编年史：建宗、每位金丹+的诞生与陨落、宗门战、天才入门、
叛逃、飞升大典……条目带时间戳与涉事弟子链接。

- 形态：竖向时间轴，按甲子分卷（"第一卷 · 草创六十年"）。
- 价值：玩家随时回看"我的宗门史"；通关/灭门时生成**全卷回顾**，这是玩家最想截图分享的东西。
- 实现成本极低（事件系统加一个 `chronicle: true` 标记），叙事回报极高。

## 5. 颜色 Token（全局唯一颜色出口）

```css
--rarity-0: #9ca3af;  /* 凡 · 白(灰) */
--rarity-1: #22c55e;  /* 灵 · 绿 */
--rarity-2: #3b82f6;  /* 玄 · 蓝 */
--rarity-3: #a855f7;  /* 地 · 紫 */
--rarity-4: #f59e0b;  /* 天 · 金 */
--rarity-5: #ef4444;  /* 圣 · 红 */
--rarity-6: linear-gradient(90deg,#f87171,#fbbf24,#34d399,#60a5fa,#a78bfa); /* 仙 · 彩 */
```

人名、物品名、边框、进度条一律从这套 token 取色；事件流类别色（机遇金/危机红/人事青）单独一小组。
